# ProyectoIntegradoPORTCARS
Proyecto Integrado de Desarrollo de Aplicaciones Multiplataforma

Este esta siendo desarrollado por Blas Villalobos Romero en el periodo de practicas del Ciclo Formativo de Grado Superior de Desarrollo de Aplicaciones Multiplataforma, este proyecto cumplimenta el modulo de Proyecto Integrado. En este se esta realizando una aplicación en Java Swing con conexiones a base de datos y uso de microservicios que se han realizado en PHP. La aplicacion consiste en la importación y exportación de vehículos.

Si desea añadir alguna sugerencia o cambio puede contactar conmigo.
